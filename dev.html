Song Deck

  • Live Site
  • Github

Song Deck is a full-stack application built to expose users to new artists and songs. It utilizes a custom algorithm to generate a random song ID which is passed into the Spotify API. The information for that song is then returned to the user and allows them to preview the track to determine if they potentially like the song. If they do, they can save the song to their account as well as navigate to the artists page on the Spotify website so that they can learn more about the artist, their releases, and upcoming shows.

    Technologies Stack:
  • React
  • Javascript
  • Firebase (Auth and Storage)
  • Spotify API
